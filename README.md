# Turn Your Screen into a Real 3D World!

Tired of the flat, 2D screen? Ever wanted to create your own 3D world, but found it overwhelmingly complex? Many give up after being beaten down by the steep learning curve.

Now, 3D-draw is here to change that!

Built upon the beginner-friendly ```easyx.h``` graphics library, this project makes 3D drawing accessible and understandable. Every line of code is crafted to be readable, helping you grasp the core concepts of 3D graphics from the ground up.

# 项目简介：3D-Draw —— 你的第一行3D代码
你是否已经看腻了扁平的二维屏幕？是否曾梦想亲手构建一个属于自己的3D世界，却被那些复杂晦涩的图形学和矩阵计算劝退？

对于初学者而言，从2D思维跃迁到3D空间常常是一道令人望而生畏的门槛。复杂的理论、庞大的渲染引擎、难以理解的底层API……这些让许多满怀热情的学习者屡屡受挫，最终选择了放弃。

现在，「3D-Draw」 项目正是为此而生。

我们并非要构建一个功能俱全的游戏引擎，而是旨在拆解3D绘制的核心原理，提供一个极致简约、清晰可见的入门路径。本项目完全基于 EasyX 图形库 开发，这是一个专为C/C++初学者设计的图形工具包。我们的每一行代码都遵循一个最高原则：拒绝黑盒，力求可读。

在这里，你将看到：

- 如何将三维坐标投影到二维屏幕（核心的透视变换）。

- 如何加载并绘制一个3D模型的obj文件。

我们坚信，理解远比调用更重要。3D-Draw 就像一个透明的教学模型，剥开了商业引擎华丽的外壳，让你直接触及3D图形最本质的骨架。它非常适合：

- 想要理解3D图形学基础概念的学生或爱好者。

- 厌倦了单纯调用API，渴望“知其所以然”的开发者。

任何怀有好奇心，想用C语言和几百行代码触碰3D奥秘的人。

从零开始，亲手“绘制”出你对三维空间的第一份理解。这个世界，不再只是屏幕上的平面幻象，而将成为你代码中可被掌控的疆域。
